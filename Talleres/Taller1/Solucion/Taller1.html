(10 pts) Explique muy brevemente y con sus propias palabras los siguientes conceptos (puede investigarlos pero al escribir trate de no tener ninguna fuente abierta).

  • Inteligencia artificial

La inteligencia artificial es un campo de investigacin y trabajo que se enfoca en construir sistemas capaces de simular la inteligencia humana, permitiendo que realicen tareas intelectuales de manera automatizada. Su objetivo principal es desarrollar tecnologas que puedan resolver problemas, aprender y tomar decisiones de forma autnoma, facilitando procesos que normalmente requieren habilidades cognitivas.

  • Machine Learning (Aprendizaje de mquinas)

El machine learning es una rama de la inteligencia artificial que se centra en desarrollar sistemas capaces de aprender relaciones, patrones y estructuras a partir de datos mediante algoritmos y modelos matemticos. Estos sistemas aprenden de los datos y ajustan sus predicciones o decisiones basndose en ellos. Sin embargo, muchas veces funcionan como una "caja negra", ya que su proceso interno es difcil de interpretar, lo que plantea desafos en trminos de transparencia y comprensin de cmo se obtienen los resultados.

  • Aprendizaje supervisado, no supervisado y reforzado

El aprendizaje supervisado, no supervisado y por refuerzo son tres enfoques principales en machine learning. En el aprendizaje supervisado, el modelo se entrena con datos etiquetados, es decir, con ejemplos que incluyen las respuestas correctas, para luego hacer predicciones sobre nuevos datos. El aprendizaje no supervisado trabaja con datos no etiquetados, donde el objetivo es encontrar patrones o estructuras ocultas en los datos, como la agrupacin o reduccin de dimensionalidad. Por su parte, el aprendizaje por refuerzo se basa en que un agente interacte con un entorno, recibiendo recompensas o castigos segn sus acciones, con el fin de aprender a maximizar recompensas a largo plazo.

  • Datos de entrenamiento, prueba y validacin

Los datos de entrenamiento, prueba y validacin son fundamentales en el desarrollo de modelos de machine learning. Los datos de entrenamiento se utilizan para ensear al modelo a aprender patrones y relaciones en los datos, ajustando sus parmetros internos. Los datos de validacin se emplean durante el entrenamiento para ajustar hiperparmetros y prevenir el sobreajuste, ayudando a evaluar el rendimiento del modelo en datos no vistos durante el entrenamiento. Finalmente, los datos de prueba se reservan para la evaluacin final del modelo, midiendo su capacidad para generalizar en nuevos datos y proporcionando una mtrica objetiva de su desempeo.

  • Mtodo de validacin cruzada

La validacin cruzada es una tcnica utilizada en machine learning para evaluar el rendimiento de un modelo de manera ms robusta y evitar el sobreajuste. Consiste en dividir el conjunto de datos en varios subconjuntos o "folds". El modelo se entrena en varios de estos subconjuntos y se prueba en los restantes, repitiendo este proceso varias veces. En cada iteracin, se utiliza un fold diferente como conjunto de prueba, mientras que los dems se emplean para el entrenamiento. Al final, los resultados se promedian para obtener una estimacin ms precisa y confiable del rendimiento del modelo en datos no vistos.

Cree una red neuronal usando neuronas sigmoides y una capa final softmax. Para esto puede usar la funcion Sequential de keras:

In[20]:
model = keras.Sequential([
    layers.Dense(512, activation ="sigmoid") ,
    layers.Dense(10, activation ="softmax")
])

En esta celda, la funcin Sequential se utiliza para crear un modelo secuencial, este modelo permite apilar capas de forma lineal, es decir, una capa tras otra.

Para este caso se crea un modelo secuencial con dos capas densas (Dense):

  • La primera capa tiene 512 neuronas con la funcin de activacin sigmoide
  • La segunda capa tiene 10 neuronas con la funcin de activacin softmax

La segunda capa de la red uso la funcin de activacin softmax se utiliza principalmente cuando el objetivo es la clasificacin multiclase. Esta funcin toma un vector de valores de entrada (que puede ser cualquier nmero real) y los convierte en probabilidades, es decir, en valores en el rango [0, 1] que suman 1.

Cada neurona de la capa softmax produce un valor que representa la probabilidad de que la entrada pertenezca a una de las clases especficas. En este caso, se generar 10 probabilidades, una para cada digito posible. La clase con la probabilidad ms alta es la prediccin del modelo.

In[21]:
model.compile(optimizer="SGD",
loss ="SparseCategoricalCrossentropy",
metrics =["accuracy"])

La funcin compile se utiliza para configurar el modelo antes de entrenarlo. Es un paso crucial que define cmo el modelo se optimizar y evaluar durante el proceso de entrenamiento. Al llamar a model.compile(), se especifican tres aspectos clave: el optimizador, la funcin de prdida y las mtricas de evaluacin. Aunque los tres parametros son claves para evaluar el modelo en el entrenamiento para este caso, es clave mencionar que la funcion de perdida usada como "SparseCategoricalCrossentropy", es adecuada para problemas de clasificacin multiclase donde las etiquetas son enteros en lugar de vectores one-hot. Esta funcin de prdida compara la probabilidad predicha para cada clase (producida por la activacin softmax) con la clase real y calcula la prdida para cada muestra.
